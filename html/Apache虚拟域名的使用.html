1. 下载页面

http://www.apachehaus.com/cgi-bin/download.plx

2. 解压到C盘根目录，管理员安装

cd \Apache24
cd bin
httpd -t
httpd -k install -n Apache

3. 打开 C:\Apache24\conf\httpd.conf，引用httpd-vhosts.conf


# Virtual hosts
Include conf/extra/httpd-vhosts.conf


4. 打开 C:\Apache24\conf\extra\httpd-vhosts.conf，分配目录域名和权限

<VirtualHost *:80>
    DocumentRoot "D:\Drive\emm0"
    ServerName local_site.com
</VirtualHost>

NameVirtualHost 127.0.0.1:80
<Directory "D:\Drive\emm0">
    AllowOverride All
    Require all granted
</Directory>

5. 打开 C:\Windows\System32\drivers\etc\hosts，加至hosts

127.0.0.1 local_site.com

6. 为 C:\Apache24\bin\ApacheMonitor.exe创建快捷方式，加入开机启动项


shell:startup

msconfig
